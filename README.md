# 🛒 Blinkit Sales Analysis | Python EDA

## 📌 Overview
Exploratory Data Analysis on Blinkit grocery sales data
using Python — covering sales trends, outlet performance,
item categories, and fat content analysis.

## 📊 Key Analysis Areas
- Total Sales, Avg Sales, Total Items, Avg Ratings KPIs
- Sales by Fat Content (Pie Chart)
- Sales by Item Type (Bar Chart)
- Sales by Outlet Location & Fat Content (Grouped Bar)
- Sales Trend by Outlet Establishment Year (Line Chart)
- Sales by Outlet Size (Pie Chart)
- Sales by Outlet Location Type (Horizontal Bar)

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (Visualizations)
- Jupyter Notebook

## 📁 Files
- `Blinkit_Analysis.ipynb` — Full analysis notebook
- `blinkit_data.csv` — Raw dataset
- `Blinkit_Analysis.pptx` — Business presentation

## 💡 Key Insights
- Analyzed sales performance across outlet sizes,
  locations, and item categories
- Identified top performing item types and outlets
- Visualized year-wise sales growth trends
- SQL Server (Data Cleaning, KPIs, PIVOT queries)
